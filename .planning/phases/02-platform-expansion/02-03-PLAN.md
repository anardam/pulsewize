ScrapeCreators API (from RESEARCH.md):
- Base URL: `https://api.scrapecreators.com`
- Auth header: `x-api-key: SCRAPECREATORS_API_KEY`
- Twitter: `GET /v1/twitter/profile?handle={handle}`
- TikTok: `GET /v1/tiktok/profile?handle={handle}`
- LinkedIn: `GET /v1/linkedin/profile?url={fullProfileUrl}`
- Facebook: `GET /v1/facebook/profile?url={fullPageUrl}`
- Timeout: 15s (AbortSignal.timeout(15000))

**src/lib/scrapers/twitter/client.ts:**
- Const `SCRAPECREATORS_BASE = "https://api.scrapecreators.com"` at top
- Zod schema `TwitterApiSchema` covering: screen_name, name, description, followers_count, friends_count, statuses_count, favourites_count, verified, is_blue_verified, profile_image_url_https — all `.optional()`
- `export async function scrapeTwitterProfile(handle: string): Promise<ScraperResult>`:
  - Read `process.env.SCRAPECREATORS_API_KEY` — error if missing
  - Strip @ from handle: `handle.replace(/^@/, "").trim()`
  - Fetch `${SCRAPECREATORS_BASE}/v1/twitter/profile?handle=${encodeURIComponent(cleanHandle)}`
  - Headers: `{ "x-api-key": apiKey }`
  - Timeout: `AbortSignal.timeout(15000)`
  - Non-ok response → `{ success: false, error: \`API error: ${res.status}\` }`
  - Zod safeParse — failure → `{ success: false, error: "Unexpected API response shape" }`
  - Map to TwitterProfile: `{ platform: "twitter", username: data.screen_name ?? cleanHandle, fullName: data.name ?? cleanHandle, biography: data.description ?? "", followersCount: data.followers_count ?? 0, followingCount: data.friends_count ?? 0, postsCount: data.statuses_count ?? 0, isVerified: data.verified ?? false, isBlueVerified: data.is_blue_verified ?? false, profilePicUrl: data.profile_image_url_https ?? "", tweetsCount: data.statuses_count ?? 0, likesCount: data.favourites_count ?? 0 }`
  - Catch block → `{ success: false, error: "Network error fetching Twitter profile" }`
